Sciety Docmaps API v1

Index

https://sciety.org/docmaps/v1/index
All available docmaps.
https://sciety.org/docmaps/v1/index?updatedAfter=2022-03-25
Docmaps updated after a given date.
https://sciety.org/docmaps/v1/index?publisheraccount=https://sciety.org/groups/biophysics-colab
Docmaps related to the evaluations of a particular group. The publisheraccount is part of each Docmap in the full index.

Single Article

https://sciety.org/docmaps/v1/articles/{DOI}.docmap.json

Example: https://sciety.org/docmaps/v1/articles/10.21203/rs.3.rs-955726/v1.docmap.json
